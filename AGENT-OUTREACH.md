# Insurance Company B2B Agent Outreach Reference
> Use this to pitch the Insurance Toolkit to agents at each company.

---

## 🏆 Best Companies to Pitch First
**Farmers, Progressive, Nationwide, Liberty Mutual** — all heavy independent agent users. Best ROI.
**State Farm & Allstate** — captive agents only, pitch goes through corporate or direct to agency owners.
**GEICO** — independent agent channel ("Broker Now") is new and small. Lower priority.
**USAA** — NO agent channel at all. Reframe as enterprise/internal tool if pursuing.
